# 3d-web

This folder is for tutorial of threejs

## Installation

```bash
# Install yarnw
yarn set version stable
yarn install 

# for vite
yarn create vite my-vue-app --template vue

yarn add -D vite # manual installation

# to run (3 methods)
yarn vite
npx vite
npm run devv

```
